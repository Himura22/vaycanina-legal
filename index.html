About

Netra Vaycanina is a private, single-user content automation system that publishes videos to the owner's own YouTube channel (@vaycanina). It is operated solely by the channel owner on a personal PC. It is not a public product, has no third-party users, and accepts no inbound sign-ups.

The application uses Google's YouTube Data API v3 strictly to upload, update metadata, and read analytics for the channel owner's own videos. No other YouTube account is accessed, read, or stored.

OAuth Scopes Requested

  • https://www.googleapis.com/auth/youtube.upload — upload videos to the owner's own channel
  • https://www.googleapis.com/auth/youtube — update metadata on the owner's own videos
  • https://www.googleapis.com/auth/youtube.readonly — read analytics on the owner's own channel

All data accessed belongs to the channel owner. The OAuth refresh token is stored only on the owner's local PC and is never transmitted to any third party. See the Privacy Policy for the full data-handling commitments.

Limited Use Compliance

Netra Vaycanina's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements. Data is used only to provide the owner-facing publishing feature; it is never sold, shared, or used for advertising.
